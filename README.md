# Text-RPG

### Install package and export package

1. Start the FastAPI Server (In venv environment)
   ```bash
   uvicorn app.main:app --reload
   ```

2. Start the local cli program
   ```bash
   python3 cli.py
   ```

### Install package and export package

1. Install package
   ```bash
   pip install -r requirements.txt
   ```

2. Export package
   ```bash
   pip freeze > requirements.txt
   ```
